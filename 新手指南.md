# React Native 新手指南

本项目使用 **Expo Router** 作为导航解决方案，基于文件系统的路由系统。

---

## 一、如何增加页面

### 1.1 在底部标签栏增加新页面

在 `app/(tabs)/` 目录下创建新的 `.tsx` 文件即可自动添加到底部标签栏。

**步骤：**

1. 在 `app/(tabs)/` 目录下创建新文件，例如 `profile.tsx`：
```tsx
// app/(tabs)/profile.tsx
import { ThemedText } from '@/components/themed-text';
import { ThemedView } from '@/components/themed-view';
import { StyleSheet } from 'react-native';

export default function ProfileScreen() {
  return (
    <ThemedView style={styles.container}>
      <ThemedText type="title">个人中心</ThemedText>
    </ThemedView>
  );
}

const styles = StyleSheet.create({
  container: {
    flex: 1,
    justifyContent: 'center',
    alignItems: 'center',
    padding: 20,
  },
});
```

2. 在 `app/(tabs)/_layout.tsx` 中配置标签：
```tsx
<Tabs.Screen
  name="profile"
  options={{
    title: '我的',
    tabBarIcon: ({ color }) => <IconSymbol size={28} name="person.fill" color={color} />,
  }}
/>
```

**重要：**
- 文件名即为路由名称（如 `profile.tsx` 路由为 `/profile`）
- `index.tsx` 是默认首页，路由为 `/`
- 图标使用 `IconSymbol` 组件，图标名称来自 SF Symbols

---

### 1.2 创建独立页面（不在标签栏）

在 `app/` 目录下直接创建文件即可创建独立页面。

**示例：**
```tsx
// app/settings.tsx
import { ThemedText } from '@/components/themed-text';
import { ThemedView } from '@/components/themed-view';
import { StyleSheet } from 'react-native';

export default function SettingsScreen() {
  return (
    <ThemedView style={styles.container}>
      <ThemedText type="title">设置页面</ThemedText>
    </ThemedView>
  );
}

const styles = StyleSheet.create({
  container: {
    flex: 1,
    padding: 20,
  },
});
```

**路由访问：** `/settings`

---

### 1.3 创建分组页面

使用 `(group)` 语法创建页面分组，括号内的组名不会出现在 URL 中。

**示例：**
```
app/
  (auth)/
    login.tsx      # 路由: /login
    register.tsx   # 路由: /register
  (tabs)/
    index.tsx      # 路由: /
    explore.tsx    # 路由: /explore
```

---

### 1.4 创建动态路由

使用 `[param]` 语法创建动态路由。

**示例：**
```tsx
// app/user/[id].tsx
import { ThemedText } from '@/components/themed-text';
import { ThemedView } from '@/components/themed-view';
import { useLocalSearchParams } from 'expo-router';

export default function UserScreen() {
  const { id } = useLocalSearchParams();

  return (
    <ThemedView>
      <ThemedText>用户ID: {id}</ThemedText>
    </ThemedView>
  );
}
```

**路由访问：** `/user/123` → `id = "123"`

## 二、如何导航

### 2.1 使用 Link 组件（声明式导航）

用于静态链接，类似 HTML 的 `<a>` 标签。

```tsx
import { Link } from 'expo-router';

<Link href="/settings">
  <ThemedText>前往设置</ThemedText>
</Link>

// 外部链接
<Link href="https://expo.dev" target="_blank">
  <ThemedText>访问 Expo</ThemedText>
</Link>
```

**带参数的链接：**
```tsx
<Link href={{ pathname: '/user/[id]', params: { id: '123' } }}>
  <ThemedText>查看用户</ThemedText>
</Link>
```

---

### 2.2 使用 router（编程式导航）

用于动态导航，适合在事件处理中使用。

```tsx
import { router } from 'expo-router';

// 导航到指定页面
router.push('/settings');

// 替换当前页面
router.replace('/settings');

// 返回上一页
router.back();

// 返回到首页
router.dismiss();

// 带参数导航
router.push({
  pathname: '/user/[id]',
  params: { id: '123' }
});
```

**完整示例：**
```tsx
import { TouchableOpacity, Text } from 'react-native';
import { router } from 'expo-router';

function NavigateButton() {
  return (
    <TouchableOpacity onPress={() => router.push('/settings')}>
      <Text>前往设置</Text>
    </TouchableOpacity>
  );
}
```

---

### 2.3 导航到模态框

在 `app/_layout.tsx` 中定义的模态页面：

```tsx
// app/modal.tsx 已在 _layout.tsx 中配置为模态
import { Button, Text, View } from 'react-native';
import { router } from 'expo-router';

export default function ModalScreen() {
  return (
    <View style={{ flex: 1, justifyContent: 'center', alignItems: 'center' }}>
      <Text>模态框页面</Text>
      <Button title="关闭" onPress={() => router.back()} />
    </View>
  );
}

// 导航方式
router.push('/modal');
```

---

### 2.4 获取导航参数

**在页面组件中获取参数：**

```tsx
import { useLocalSearchParams } from 'expo-router';

export default function UserScreen() {
  const { id, name } = useLocalSearchParams();

  return (
    <ThemedView>
      <ThemedText>用户ID: {id}</ThemedText>
      <ThemedText>用户名: {name}</ThemedText>
    </ThemedView>
  );
}
```

## 三、如何增加底部按钮

### 3.1 修改现有底部标签

编辑 `app/(tabs)/_layout.tsx`：

```tsx
import { Tabs } from 'expo-router';
import { IconSymbol } from '@/components/ui/icon-symbol';
import { Colors } from '@/constants/theme';
import { useColorScheme } from '@/hooks/use-color-scheme';

export default function TabLayout() {
  const colorScheme = useColorScheme();

  return (
    <Tabs
      screenOptions={{
        tabBarActiveTintColor: Colors[colorScheme ?? 'light'].tint,
        headerShown: false,  // 是否显示顶部导航栏
      }}>
      
      {/* 首页 */}
      <Tabs.Screen
        name="index"
        options={{
          title: '首页',
          tabBarIcon: ({ color }) => <IconSymbol size={28} name="house.fill" color={color} />,
        }}
      />
      
      {/* 探索页 */}
      <Tabs.Screen
        name="explore"
        options={{
          title: '探索',
          tabBarIcon: ({ color }) => <IconSymbol size={28} name="paperplane.fill" color={color} />,
        }}
      />
      
      {/* 新增：个人中心 */}
      <Tabs.Screen
        name="profile"
        options={{
          title: '我的',
          tabBarIcon: ({ color }) => <IconSymbol size={28} name="person.fill" color={color} />,
        }}
      />
    </Tabs>
  );
}
```

---

### 3.2 自定义底部标签样式

```tsx
<Tabs
  screenOptions={{
    tabBarActiveTintColor: '#007AFF',           // 选中文字颜色
    tabBarInactiveTintColor: '#999999',         // 未选中文字颜色
    tabBarStyle: {
      backgroundColor: '#FFFFFF',               // 背景色
      borderTopWidth: 1,
      borderTopColor: '#E0E0E0',                // 顶部边框
      height: 60,                                // 高度
      paddingBottom: 8,                          // 底部内边距
    },
    tabBarLabelStyle: {
      fontSize: 12,
      fontWeight: '600',
    },
  }}>
```

---

### 3.3 使用自定义图标

除了 `IconSymbol`，还可以使用其他图标库：

**使用 @expo/vector-icons：**
```tsx
import { Ionicons } from '@expo/vector-icons';

<Tabs.Screen
  name="profile"
  options={{
    title: '我的',
    tabBarIcon: ({ color, size }) => (
      <Ionicons name="person-circle" size={size} color={color} />
    ),
  }}
/>
```

**使用自定义图片：**
```tsx
import { Image } from 'react-native';

<Tabs.Screen
  name="profile"
  options={{
    title: '我的',
    tabBarIcon: ({ color, focused }) => (
      <Image
        source={require('@/assets/images/profile.png')}
        style={{
          width: 28,
          height: 28,
          tintColor: color,
        }}
      />
    ),
  }}
/>
```

---

### 3.4 隐藏特定页面的底部标签

```tsx
<Tabs.Screen
  name="detail"
  options={{
    title: '详情',
    tabBarButton: () => null,  // 隐藏底部标签
  }}
/>
```

## 四、完整示例

### 创建一个新的标签页面并配置导航

**步骤 1：创建页面文件**
```tsx

import { ThemedText } from '@/components/themed-text';
import { ThemedView } from '@/components/themed-view';
import { TouchableOpacity, StyleSheet } from 'react-native';
import { router } from 'expo-router';

export default function WalletScreen() {
  return (
    <ThemedView style={styles.container}>
      <ThemedText type="title">我的钱包</ThemedText>
```

## 五、常用技巧

### 5.1 页面间传值

**方式一：通过 URL 参数**
```tsx
// 发送
router.push('/user/[id]', { params: { id: '123' } });

// 接收
const { id } = useLocalSearchParams();
```

**方式二：通过路由状态**
```tsx
// 发送
router.push({
  pathname: '/detail',
  params: { data: JSON.stringify({ name: 'Alice', age: 25 }) }
});

// 接收
const { data } = useLocalSearchParams();
const parsedData = JSON.parse(data as string);
```

---

### 5.2 防止重复点击

```tsx
import { useCallback } from 'react';

function Button() {
  const handlePress = useCallback(() => {
    if (!router.canGoBack()) {
      // 不能返回
      return;
    }
    router.back();
  }, []);

  return <TouchableOpacity onPress={handlePress}>返回</TouchableOpacity>;
}
```

---

### 5.3 添加加载动画

```tsx
import { useFocusEffect } from 'expo-router';
import { useState } from 'react';

export default function Screen() {
  const [loading, setLoading] = useState(true);

  useFocusEffect(
    useCallback(() => {
      setLoading(false);
      return () => {
        setLoading(true);  // 页面离开时重置
      };
    }, [])
  );

  if (loading) return <ThemedText>加载中...</ThemedText>;

  return <ThemedView>页面内容</ThemedView>;
}
```

## 六、常见问题

### Q1：页面没有显示在底部标签栏？
**A：** 确保：
1. 页面文件在 `app/(tabs)/` 目录下
2. 在 `app/(tabs)/_layout.tsx` 中已配置 `<Tabs.Screen>`
3. 文件名与 `name` 属性一致

---

### Q2：导航没有效果？
**A：** 检查：
1. 路由路径是否正确（区分大小写）
2. 目标页面文件是否存在
3. 是否使用了正确的导航方式（`router.push` 或 `Link`）

---

### Q3：如何隐藏顶部导航栏？
**A：** 在 `screenOptions` 中设置 `headerShown: false`：
```tsx
<Tabs
  screenOptions={{
    headerShown: false,
  }}
/>
```

---

### Q4：如何修改底部标签栏位置？
**A：** Expo Router 的底部标签栏默认固定在底部，不支持修改位置。
