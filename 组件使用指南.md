## 一、ParallaxScrollView 是什么

### 1.1 什么是 ParallaxScrollView

**ParallaxScrollView**（视差滚动视图）是一个带视差效果的滚动容器组件。当用户滚动页面时，顶部图片会产生视差动画效果：
- 向上滚动时，图片会放大并产生位移
- 向下滚动时，图片会缩小并跟随移动
- 创造出有层次的视觉体验

**实现原理：**
- 基于 `react-native-reanimated` 实现平滑动画
- 通过监听滚动偏移量，动态计算图片的变换（位移、缩放）
- 高性能，使用原生动画优化

---

### 1.2 ParallaxScrollView 使用方法

**基础用法：**
```tsx
import { Image } from 'expo-image';
import ParallaxScrollView from '@/components/parallax-scroll-view';
import { ThemedText } from '@/components/themed-text';
import { ThemedView } from '@/components/themed-view';

export default function MyScreen() {
  return (
    <ParallaxScrollView
      headerBackgroundColor={{ light: '#A1CEDC', dark: '#1D3D47' }}
      headerImage={
        <Image
          source={require('@/assets/images/partial-react-logo.png')}
          style={{ width: '100%', height: 250 }}
        />
      }>
      <ThemedView>
        <ThemedText type="title">页面标题</ThemedText>
        <ThemedText>页面内容...</ThemedText>
      </ThemedView>
    </ParallaxScrollView>
  );
}
```

---

### 1.3 ParallaxScrollView 属性说明

| 属性 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `headerImage` | `ReactElement` | 是 | 顶部图片或内容 |
| `headerBackgroundColor` | `{ light: string, dark: string }` | 是 | 头部背景色（支持深色模式） |
| `children` | `ReactNode` | 是 | 页面内容 |

---

### 1.4 自定义 ParallaxScrollView

**调整头部高度：**
```tsx
// 在 parallax-scroll-view.tsx 中修改
const HEADER_HEIGHT = 350;  // 默认是 250
```

**改变动画效果：**
```tsx
// 在 parallax-scroll-view.tsx 中修改插值逻辑
const headerAnimatedStyle = useAnimatedStyle(() => {
  return {
    transform: [
      {
        translateY: interpolate(
          scrollOffset.value,
          [-HEADER_HEIGHT, 0, HEADER_HEIGHT],
          [-HEADER_HEIGHT / 3, 0, HEADER_HEIGHT * 0.5]  // 调整位移范围
        ),
      },
      {
        scale: interpolate(
          scrollOffset.value,
          [-HEADER_HEIGHT, 0, HEADER_HEIGHT],
          [1.5, 1, 1]  // 调整缩放范围
        ),
      },
    ],
  };
});
```

### 7.2 创建一个表单页面

```tsx
// app/(tabs)/form.tsx
import { useState } from 'react';
import {
  View,
  TextInput,
  TouchableOpacity,
  StyleSheet,
  Alert,
} from 'react-native';
import { ThemedText } from '@/components/themed-text';
import { ThemedView } from '@/components/themed-view';
import ParallaxScrollView from '@/components/parallax-scroll-view';

export default function FormScreen() {
  const [username, setUsername] = useState('');
  const [email, setEmail] = useState('');
  const [password, setPassword] = useState('');
  const [loading, setLoading] = useState(false);

  const handleSubmit = () => {
    if (!username || !email || !password) {
      Alert.alert('提示', '请填写所有字段');
      return;
    }

    setLoading(true);
    setTimeout(() => {
      setLoading(false);
      Alert.alert('成功', '提交成功！');
    }, 2000);
  };

  return (
    <ParallaxScrollView
      headerBackgroundColor={{ light: '#A1CEDC', dark: '#1D3D47' }}
      headerImage={<View style={styles.header} />}>
      <ThemedView style={styles.container}>
        <ThemedText type="title" style={styles.title}>注册</ThemedText>

        <TextInput
          style={styles.input}
          placeholder="用户名"
          value={username}
          onChangeText={setUsername}
          autoCapitalize="none"
        />

        <TextInput
          style={styles.input}
          placeholder="邮箱"
          value={email}
          onChangeText={setEmail}
          keyboardType="email-address"
          autoCapitalize="none"
        />

        <TextInput
          style={styles.input}
          placeholder="密码"
          value={password}
          onChangeText={setPassword}
          secureTextEntry
        />

        <TouchableOpacity
          style={[styles.button, loading && styles.buttonDisabled]}
          onPress={handleSubmit}
          disabled={loading}>
          <ThemedText style={styles.buttonText}>
            {loading ? '提交中...' : '提交'}
          </ThemedText>
        </TouchableOpacity>
      </ThemedView>
    </ParallaxScrollView>
  );
}

const styles = StyleSheet.create({
  header: {
    height: 250,
    backgroundColor: '#A1CEDC',
  },
  container: {
    padding: 20,
    gap: 16,
  },
  title: {
    textAlign: 'center',
    marginBottom: 20,
  },
  input: {
    borderWidth: 1,
    borderColor: '#ccc',
    borderRadius: 8,
    padding: 15,
    fontSize: 16,
    backgroundColor: '#fff',
  },
  button: {
```

## 八、常见问题

### Q1：组件找不到怎么办？
**A：**
1. 检查导入路径是否正确（使用 `@/` 别名）
2. 确认组件文件是否存在
3. 运行 `npm install` 确保依赖已安装

---

### Q2：样式不生效？
**A：**
1. 确认 `StyleSheet.create` 正确使用
2. 检查是否有样式覆盖
3. 使用 React DevTools 查看样式

---

### Q3：如何调试组件？
**A：**
1. 使用 `console.log` 打印数据
2. 使用 React DevTools 检查组件状态
3. 启用开发者菜单：iOS (Cmd+D)、Android (Cmd+M)
